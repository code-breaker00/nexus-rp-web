Préparation de la machine

Premièrement, vous allez acheter la machine à smoothies et la déployer.
Pour ce faire, vous devez l'acheter dans le F4 (Entitées), et, une fois la machine apparue, vous devrez cliquer sur ce bouton :

Différentes interfaces

Une fois la machine prête, vous aurez alors sur l'écran 3 interfaces, les voici :

Première interface

Elle sert à ranger la machine si vous voulez la déplacer

Deuxième interface

Elle sert à fabriquer les smoothies

Troisième interface

Le troisième interface sert à gérer le stockage de votre machine

Confection des smoothies

Dans cette troisième partie, nous allons voir comment créer et customiser vos smoothies
Avant toute chose, vous devrez veillez à avoir assez de matières pour confectionner vos smoothies

  • Pour remplir votre machine, vous devrez acheter vos cartons de fruits dans le F4 (Entitées)
  • Ensuite vous devrez les prendre au gravity gun, et simplement les rentrer dans la machine
  • Pour visualiser votre stock, vous devrez vous rendre dans la troisième interface (https://prnt.sc/x3212g)


Une fois votre machine alimentée en fruits, vous allez donc cliquer sur la deuxième interface (https://prnt.sc/x31ymw) afin d'ouvrir le menu de création

Vous aurez ensuite plusieurs possibilités de smoothies :

 
 
Voici un exemple
 
 
Nous allons prendre le smoothie "Bombe de Vitamine"
Comme le montre l'écran, vous aurez besoin de :
  • 1 pastèque
  • 3 bananes
  • 1 noix de coco
  • 1 grenade
  • 1 fraise
  • 2 kiwis
  • 1 citron
  • 2 oranges
  • 2 pommes

Si vous avez tout ça dans votre machine, c'est nickel on va pouvoir commencer, sinon retournez au début de cette troisième partie

Vous cliquerez donc sur le smoothie "Bombe de Vitamine".
Ensuite, vous aurez une garniture à choisir parmi celles-ci :

Voici le détail de toutes les garnitures

 
Je vais donc cliquer par exemple sur la garniture "café"
Vous allez donc prendre une coupe en appuyant sur "E" à cet endroit
 
 
Ensuite, vous allez prendre votre "Knife" (dans votre barre d'arme n°3) et vous allez couper les fruits
Cela peut être long si votre composition contient beaucoup de fruits

 
Ensuite, comme indiqué sur l'écran de la machine, vous aurez à choisir entre 3 diluants
 
 
Ensuite, on lance le mixeur en faisant appuyant à cet endroit
 
 
Une fois cela fait, votre smoothie va passer de l'autre côté de la machine, il est prêt à être dégusté !

Vous (ou le consommateur) aurez juste à appuyer sur "E" dessus pour avoir cet pop-up
 
 
Une fois acheté, le smoothie est consommable !
Le consommateur verra donc sa jauge de faim augmentée (en fonction de ce que fourni le smoothie, c'est visible sur https://prnt.sc/x35551 par exemple où on voit "Energie : +64")
Il recevra également les effets temporaires du smoothie en fonction de la garniture choisie.

Remarques : Le prix de vente du smoothie est modifiable (jusqu'à 10 000$ par smoothie) en cliquant ici
 
 
Vous pouvez préparer plusieurs smoothies et les laisser sur votre stand (https://prnt.sc/x364dj), ne vous inquiétez pas, si quelqu'un veut le prendre pour le boire, il devra vous l'acheter

Voilà ! Vous savez maintenant comment faire de bons smoothies !

Les tables légales

Les travailleurs légaux

Ces tables sont strictement LEGAL, donc les gendarmes ne peuvent pas vous arrêter si vous possédez ce type de table.
Ne pas oublier que les travailleurs légaux rapporte moins que les travailleurs illégaux.

Comment utiliser le métier ?

Lorsque vous ferez spawn vos tables et travailleurs, ils seront légaux.

Pour modifier une table légal en table illégal, il faut lui donner une "crime box"

Pour modifier un travailleur légal en travailleur illégal, il faut lui donner un "crime book"

Une fois ces éléments ajouté, vos tables auront une couleur rouge

Idem pour vos travailleurs

Votre travail sera donc illégal et vous serez susceptible de vous faire arrêter par la gendarmerie

Assurez vous que le type de table soit cohérant avec la meilleur compétence de votre travailleur

Si vous voulez changer les compétences de vos travailleurs, il vous suffira de leurs donner un "good book" puis leurs redonner un "crime book" (la génération des compétences est totalement aléatoire)

Si vous voulez changer le type de vos tables, il vous suffira de leurs donner une "good box" puis leurs redonner une "crime box" (le type de table est totalement aléatoire)

A notez qu'avec le temps, la compétence avec laquelle votre travailleur travail s'améliore

Début du travail

On peux voir sur la table, deux barres de progression, la barre "Working Progress" et la barre "Product Progress"

Vos employés évoluent jusqu'a atteindre un seuil où la barre "Working Progress" ne vous servira plus à rien car à chaque seconde la barre "Product Progress" avancera

Lorsque la barre "Product Progress" est complète, un élément que vous pourrez récupérer apparaitra et vous donnera de l'argent

Vous avez désormais de bons employés prêt à travailler pour vous ! Bon jeu !
